# Test Summary

## Project

**Automotive Wireless Sensor Prototyping Kit**

Master's Thesis  
University of Bremen

Developed in collaboration with **HELLA (FORVIA)**

---

## Validation Highlights

The developed prototype successfully demonstrated:

- ✅ Stable sensor acquisition
- ✅ Pressure sensor validation
- ✅ CAN communication
- ✅ Embedded HTTP web server
- ✅ Wi-Fi dashboard for live monitoring
- ✅ SD card data logging
- ✅ Offline MATLAB-based data analysis

---

## Development Outcome

The project resulted in a functional embedded platform capable of:

- Monitoring multiple automotive sensors
- Transmitting measurements over CAN
- Displaying real-time data through a web dashboard
- Logging sensor data for offline analysis
- Supporting rapid firmware prototyping and validation

---

## Repository Note

This repository contains a simplified overview of the project.

The original firmware, hardware design files, communication protocols, and other proprietary implementation details are not included due to confidentiality agreements established during the collaboration with **HELLA (FORVIA)**.
